Thus repo is to help Cecile Babiole with a RAVE model of her voice for her project Tissus de diableries

Step 1:
Download nn~ for PD from 
https://github.com/acids-ircam/nn_tilde
On the right hand side, please see Releases. As of 28 May 2026, it was v1.6.0
https://github.com/acids-ircam/nn_tilde/releases/tag/v1.6.0
Make sure to select the right operating system (i.e. Linux) and the right CPU hardware (x86) and download the .tar.gz archive file and decompress.

Then follow install instructions found at the top level
 https://github.com/acids-ircam/nn_tilde
 
 Once nn~ is installed, then open the PD patch "babiole-test.pd"
 
 Audio input is on the left - this can be a microphone.
 You can also play the model with a sound file. A good way to test is using the audio that the model was trained on.
 
 The patch allows you to listen to the source (ADC or Test sound), and listen to the Rave model.
 
 There are faders that let you navigate in 8 dimensions of the latent space. This allows strange sounds to emerge, even when the original training data is used as input.
 
 There is a Master Volume control on the right.
